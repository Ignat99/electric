Package for handling the simulation tool.

The {@link com.sun.electric.tool.simulation.Simulation Simulation} class is the "tool" which represents all work done with simulation data. An {@link com.sun.electric.tool.simulation.Engine Engine} is something which produces simulation data. All the other classes in this package deal with that data.

The simulation data seen in a particular waveform window are encapsulated by a corresponding instance of {@link com.sun.electric.tool.simulation.Stimuli Stimuli}. An instance of {@link com.sun.electric.tool.simulation.Stimuli Stimuli}, in turn, consists of one or more {@link com.sun.electric.tool.simulation.Analysis Analysis} instances, each of which contains one or more {@link com.sun.electric.tool.simulation.Signal Signal}s. A {@link com.sun.electric.tool.simulation.Signal Signal} contains one or more {@link com.sun.electric.tool.simulation.Waveform Waveform}s, one for each {@linkplain com.sun.electric.tool.user.waveform.SweepSignal sweep}.

{@link com.sun.electric.tool.simulation.Signal Signal}s are grouped into {@link com.sun.electric.tool.simulation.Analysis Analyses} according to the kind of signal ({@link com.sun.electric.tool.simulation.AnalogSignal AnalogSignal} or {@link com.sun.electric.tool.simulation.DigitalSignal DigitalSignal}) and {@linkplain com.sun.electric.tool.simulation.Analysis.AnalysisType type} of analysis being performed ({@linkplain com.sun.electric.tool.simulation.Analysis.AnalysisType#ANALYSIS_SIGNALS signal}, {@linkplain com.sun.electric.tool.simulation.Analysis.AnalysisType#ANALYSIS_AC AC}, {@linkplain com.sun.electric.tool.simulation.Analysis.AnalysisType#ANALYSIS_DC DC}, {@linkplain com.sun.electric.tool.simulation.Analysis.AnalysisType#ANALYSIS_TRANS transient}, or {@linkplain com.sun.electric.tool.simulation.Analysis.AnalysisType#ANALYSIS_MEAS measurement}).
